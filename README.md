# Fake News and Deepfake Detection System

## Project Overview

This project detects:

- Fake News
- Deepfake Images
- Deepfake Videos

It also supports:

- Fake Image Generation
- Fake Video Generation
- Performance Evaluation
- PDF Reports
- Excel Reports
- Prediction History Tracking

---

## Installation

```bash
pip install -r requirements.txt
```

## Initialize Database

```bash
python database/init_db.py
```

## Run Project

```bash
python run.py
```

## Open Browser

```text
http://127.0.0.1:5000
```
